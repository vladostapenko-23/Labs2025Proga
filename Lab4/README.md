# Лабораторна робота №4: Асинхронна детекція обличчя

## Архітектура (Рівень 2)
- **Main Thread (UI)**: Захоплює відео з камери, малює рамки та відображає картинку. Не блокується нейромережею.
- **Worker Thread (FaceDetector)**: Окремий потік для інференсу DNN. Використовує `std::mutex` та `std::condition_variable` для обміну даними.

## Інструкція зі збірки
1. `mkdir build && cd build`
2. `cmake ..`
3. `make`
4. `./app`
