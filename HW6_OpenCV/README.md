# HW6: Simple OpenCV Camera App

Проста програма, яка зчитує відео з камери, малює прямокутник та виводить текст.

## Інструкція для запуску (Build & Run)

1. Встановіть залежності:
   `sudo apt install libopencv-dev cmake g++`

2. Скомпілюйте проект:
   ```bash
   mkdir build
   cd build
   cmake ..
   make
   ```

3. Запустіть:
   `./SimpleCam`

Для виходу натисніть **ESC** або **q**.
